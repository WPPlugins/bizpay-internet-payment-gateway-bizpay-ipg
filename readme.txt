== Installation ==

1. Ensure you have latest version of WooCommerce plugin installed (WooCommerce 2.0+)
2. Unzip and upload contents of the plugin to your '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Configuration ==

1. Visit the WooCommerce > Settings > Checkout tab.
2. Click on *BizPay* to edit the settings. If you do not see *BizPay* in the list at the top of the screen make sure you have activated the plugin in the WordPress Plugin Manager.
3. Enable the Payment Method, name it "BizPay - Visa/MasterCard/EzCash" (this will show up on the payment page your customer sees).
4. Add in your "Merchant", "Api Key" and "Api Token" as provided by the BizPay.
6. Select "Redirect url" (URL you want BizPay to redirect after payment).
7. Click Save.
